# 智慧招采通知识库目录

本目录用于存放智慧招采通运行所需的基础素材。当前先给出说明框架，后续由商务或标书负责人填充真实文件。

## 必备知识包

1. 企业资质材料（营业执照、专业资质证书、安全生产许可等）
2. 人员证书与团队概况
3. 项目案例库（含合同、验收报告和业主评价）
4. 历史应答文件库
5. 应答文件模板库
6. 技术方案素材库
7. 商务承诺与合同条款素材
8. 常见否决风险清单
9. 应标经验复盘记录
10. 技术规范或需求任务书文本
11. 评审模型和无偏离索引示例
12. 应答文件表达禁用词库
13. 图表样式和历史图表素材
14. 政府采购法规与地方交易平台规范摘要

## 重要要求

- 资质、业绩、人员信息必须真实可核验。
- 报价和商务承诺须经决策人确认。
- 演示示例使用脱敏或虚构示范项目，不使用真实客户敏感数据。
- 参考应答文件仅供结构借鉴，不可直接继承项目名称、地点、特定参数、工期、报价和示范点内容。
- 正式应答文本中不可保留内部评审策略用语、夸张修辞和模板化机械表达。
- 遵守《中华人民共和国政府采购法》关于信息公开和公平竞争的有关规定。
